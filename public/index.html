Beginners - opdrachten
  1. Bereken de gemiddelde waarde van de spellen.
  2. Toon deze in de 'avarage value'-card.
  3. Maak een nieuwe card om de meest waardevolle game te tonen.
  4. Maak een nieuwe card om de game te tonen met het meeste gebruikers.
  5. Bedenk welke zinvolle informatie je nog meer kan halen uit de database en maak daarvoor een nieuwe kaart.
  6. Vraag je docent naar feedback en schrijf hierover een retrospective in de readme
